# Contributing

1. Fork and clone the repository  
2. Set up the environment and install dependencies:

```bash
python -m venv venv
source venv/bin/activate  # or venv\Scripts\activate on Windows
pip install -r requirements.txt
```
3. Create a new branch:

```Bash
git checkout -b my-branch-name
```
4. Make your changes and test them locally:

```Bash
python src/bot.py
```

5. Ensure your code is clean and readable.

6. Push to your fork and submit a pull request.

7. Take a moment and wait for your pull request to be reviewed and merged 🚀

## Tips for a successful pull request

- Write clear and focused changes
- Keep your changes as small as possible
- If changes are unrelated, submit separate pull requests
- Make sure your changes do not break existing behavior
